1. What is the difference between flexbox and grid?

Flexbox is a layout mode in CSS that arranges elements in a one-dimensional layout, while Grid is a layout mode that arranges elements in a two-dimensional layout. Flexbox is used for aligning items in a single direction, either horizontally or vertically, while Grid is used for creating grid-based layouts.

2. What is the purpose of media queries?

Media queries are used in CSS to apply different styles to a website or web application based on the characteristics of the device or screen displaying it, such as screen size, resolution, and screen orientation. They allow for creating responsive designs that adapt to the different devices and screen sizes that a website may be viewed on.

3. How does CSS box modal work?

CSS box model is the foundation of layout in CSS. It defines how the elements of a webpage are rendered and takes into account the element's content, padding, borders, and margins. Every HTML element is a rectangular box and the CSS box model describes how that box is calculated and how it interacts with other elements on the page. The content area is the innermost part of the box, surrounded by padding, then a border, and then margins.
